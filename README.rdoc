== Welcome

SRS GAME is written in the Ruby[link:http://ruby_lang.org/] programming language.

== Installation

    $ git clone git@github.com:jacksonwillis/srs_game.git
    $ gem install bundler[link:http://gembundler.com/]
    $ bundle install

== Playing the game with Rake

To play, use Rake on the command line:
    $ rake play:tamera   # Play as the goddess Tamera
    $ rake play:tia      # Play as a follower of the Cult of Tia

=== Command-line Arguments

srs_game takes arguments[link:http://rdoc.info/github/jacksonwillis/srs_game/master/SRSGame/Settings.default_settings] on the command line. For example:
    $ rake play:tamera SAYS_HOWDY_PARTNER=true

== Scripting

Try something like this in a ruby script:
    require "srs_game"
    require "srs_game/tia"
    SRSGame.play SRSGame::Tia

To use another game module, try something like this:
    module YourGameModule
      #...
    end

    require "srs_game"
    SRSGame.play YourGameModule
